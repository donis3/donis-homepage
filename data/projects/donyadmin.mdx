## Simple Responsive Admin Dashboard

An easy to customize admin dashboard template created with Next.js and tailwind. 
A starting point for new projects. 
It consists of a landing page, custom error page and admin dashboard.

## Features

-   Fully responsive
-   Dark mode detection and theme/dark mode preference saving in local storage
-	Shipped with 2 color themes and easy to add more.
- 	Multiple components including handcrafted data table with pagination and persistence.
-	Multiple Custom hooks including persistent state
-	Format numbers and dates according to client browser locale
- 	Responsive chart component powered by [Recharts](https://recharts.org)
