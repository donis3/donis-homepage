Costotus is a client-side cost calculator designed to help small businesses manage their finances and production costs effectively. Built with React, it offers a suite of tools to track expenses, manage inventory, and calculate product profitability.

## Features

- **Company Expenses**: Track recurring expenses like rent, salaries, and utilities.
- **Material Management**: Organize raw materials and packaging inventory.
- **Recipe Builder**: Combine materials to create recipes and determine base costs.
- **Product Costing**: Calculate precise cost-per-unit for products based on recipes.
- **Financial Insights**: View cost charts over time to analyze spending trends.
- **Multi-Currency**: Auto-fetch exchange rates for multi-currency support.
- **Productivity**: Built-in todo list widget on the dashboard.
- **Localization**: Available in English and Turkish.

## Technical Details

This Single Page Application (SPA) was bootstrapped with Create React App and runs entirely in the browser.

- **Framework**: React 17 with React Router 6 for navigation.
- **State Management**: React Context API.
- **UI/UX**: Styled with DaisyUI.
- **Charts**: Data visualization using Chart.js.
- **I18n**: Internationalization support via i18next.
